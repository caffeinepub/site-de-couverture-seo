# Specification

## Summary
**Goal:** Enrich the Verdier Couverture website with geo-targeted SEO content covering all neighboring cities in Val-d'Oise and roofing service keywords.

**Planned changes:**
- Update `frontend/index.html` meta title, meta description, and Open Graph tags with expanded keywords covering all listed neighboring cities and roofing services.
- Update the JSON-LD LocalBusiness structured data in `frontend/index.html` to add an `areaServed` array (City objects for all listed municipalities) and a `knowsAbout` array of roofing service keywords in French.
- Add a visible "Zones d'intervention" section between the Contact section and the Footer, listing all served cities in a responsive tag-cloud or grid layout consistent with the earthy terracotta/slate theme, using an H2/H3 heading.
- Update Hero, About, and Contact section text to naturally weave in references to neighboring cities and the Val-d'Oise service area in French.

**User-visible outcome:** The site displays a "Zones d'intervention" section listing all served cities, and the page content, meta tags, and structured data are enriched with geo-targeted keywords — improving local SEO visibility across the Val-d'Oise region without altering the existing design, contact details, or color scheme.
